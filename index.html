Modelo de Regresión Lineal para IMC

Descripción del Proyecto

Este proyecto utiliza regresión lineal para predecir el Índice de Masa Corporal (IMC) basado en datos sintéticos de peso y altura.

Fórmula del IMC:
IMC = peso (kg) / (altura (m)²)

El modelo fue entrenado con 100 muestras sintéticas y alcanzó un R² de 0.96, demostrando alta precisión.
